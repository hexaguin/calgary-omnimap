About Calgary Omnimap

Calgary Omnimap is a unified map of Calgary, designed to get you from point A to point B, whether you're walking around the Plus 15, riding a bike along the Glenmore Reservoir, or driving across town. Omnimap is built to work on everything from 4K TVs to budget smartphones.

Omnimap updates all of its data in real time (or as quickly as the data source allows), meaning it can also be used as a live dashboard.

Data sources

Omnimap pulls data from the following sources:

Overlay Data

  • Open Calgary
  • 511AB
  • OpenStreetMap Overpass

Basemaps

  • Mapbox
  • CARTO
  • Thunderforest
  • OpenStreetMap
  • Esri World Imagery

Technologies Used

Omnimap is built on the following technologies:

  • Mapbox Navigation Service
  • Leaflet
    • Leaflet Realtime
    • Leaflet Awesome Markers
    • Leaflet MarkerCluster
    • Leaflet Control Layers Tree
    • Leaflet Routing Machine
    • Leaflet FeatureGroup SubGroup
    • Leaflet Easy Button
  • JQuery
  • Slick
  • Bootstrap
  • Font Awesome
